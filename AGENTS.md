# Agents

Use `mise` for all toolchain and task management. Do not use a Makefile.

## Tasks

| Task | Command |
|---|---|
| Build binary | `mise run build` |
| Lint | `mise run lint` |
| Test | `mise run test` |
| Full CI | `mise run check` |
| Clean | `mise run clean` |

## Toolchain

Defined in `mise.toml`. Run `mise` commands from the project root.
